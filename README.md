## Remove Accessibility extension
Gnome Shell Extension to remove the Accessibility button in Gnome Shell 45

Based on [minzords's](https://github.com/AVAtarMod/RemoveAccessibility) extension.

Also, this version conform [extensions guidlines](https://gjs.guide/extensions/review-guidelines/review-guidelines.html) better than original version.
